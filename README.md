
# Proyecto Sistema Estudiantes

Proyecto de CRUD completo realizado en Java con Spring, Spring boot y MySql.
Permite el listado, agregación, modificación, y eliminación de registros.
Se ejecuta en consola.




## Funcionalidades

 - Listar registros
 - Agregar registros
 - Modificar registros
 - Eliminar registros




## Tecnologías

 - Java
 - Spring
 - Spring boot
 - MySql
